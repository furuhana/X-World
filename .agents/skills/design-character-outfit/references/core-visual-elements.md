# Core Visual Element System

Use this process after establishing visual sources and before selecting garments. Build a reusable visual element system that can organize clothing and accessories without reducing the design to repeated surface decoration.

## Contents

- Distinguish the layers
- Derive candidate elements
- Define the element grammar
- Translate the element into clothing
- Plan full-body deployment
- Test element quality and failure conditions
- Produce reviewable output

## Distinguish the Layers

- **Visual source:** an idea that may inspire design, such as an occupation, flower, lava flow, tool, building, ritual, personality trait, combat method, or geometric form.
- **Core visual element:** a concrete, recognizable visual motif derived from one or more sources, such as an elongated cross-star, split ring, stepped arch, or hooked crescent.
- **Application method:** how the element enters clothing, such as a panel division, closure, buckle, embroidery, raised applique, or hanging ornament.
- **Deployment plan:** where each application appears, how prominent it is, and which areas remain quiet.

Do not report a broad source such as `medical`, `ocean`, `authority`, or `flower` as the finished core element. Refine it into a motif whose visible form and recognition rules can be described.

## Derive Candidate Elements

Generate multiple candidate elements from any relevant source:

- profession, tools, procedures, and working environment
- culture, architecture, writing systems, heraldry, and ritual objects
- identity, habits, personal history, and social role
- movement, combat method, power, and equipment
- natural objects, natural processes, and abstract geometry

Sources are not limited to nature or literal symbols. Prefer an element that belongs to the character, supports the intended first impression, and can create several useful clothing decisions.

Select one primary element. Add one supporting element only when it improves contrast or specificity without competing with the primary element.

## Define the Element Grammar

Before applying the primary element, define:

- **standard form:** its clearest complete appearance
- **recognition anchors:** the smallest set of proportions, directions, intersections, gaps, or terminals that must remain recognizable
- **allowed transformations:** useful changes that preserve identity
- **forbidden transformations:** changes that turn it into a different motif or weaken recognition
- **suitable visual roles:** whether it works best as useful decorative construction, hardware, reinforcement, ornament, or another role

Useful transformations include:

- stretch, compress, rotate, invert, mirror, or repeat
- crop, interrupt, split, offset, or recombine
- simplify into an outline, intersection, axis, terminal, or layered edge
- convert into a pocket, reinforcement panel, panel boundary, seam path, closure, hardware shape, raised form, or hanging object

Do not require every application to show the complete element. Recognition may come from a stable proportion, intersection, direction, or terminal shape.

## Translate the Element into Clothing

Use different application methods according to garment location and purpose. Prefer applications that are both decorative and useful:

- **useful decorative construction:** shaped pocket, reinforced stress panel, protective overlay, adjustable tab, belt loop, hanging loop, equipment mount, or layered edge
- **silhouette:** hem, shoulder, sleeve, layered edge, or overall outer contour
- **structural:** reinforced panel division, seam path, layered boundary, false-layer construction, or attached overlay
- **functional:** buckle, clasp, zipper pull, connector, fastening plate, closure, adjuster, or mount
- **three-dimensional:** raised applique, folded form, padded shape, or independent ornament
- **planar:** embroidery, print, patch, or simplified mark
- **accessory and wearing relationship:** ear-area wearing, neck-and-chest suspension, cross-body connection, shoulder-and-back carrying, handheld or arm attachment, hip-and-leg wrapping or suspension, or an external silhouette-changing object
- **implicit:** seam direction, interrupted line, cropped fragment, layered reveal, or rhythm

Choose an application because it suits the location. Do not place a complete motif on every available garment part.

Practical use must come from the character's actual wearing, carrying, protection, adjustment, movement, or occupational needs. Do not invent unnecessary pockets, straps, clasps, mounts, or protective pieces merely to make an element appear functional.

Do not let examples of pockets, belt loops, clasps, mounts, or hardware bias all accessory applications toward the waist. Explore multiple body regions and wearing relationships before selecting an application.

High-visual-weight ornaments such as gems, ornate brooches, large metal fittings, and luminous cores are low-probability element applications. A conspicuous central chest application must have a visible attachment or load-bearing structure, a strong source, a visual responsibility proportionate to its weight, at least one structural echo, and one restrained visual echo. Do not repeat the same conspicuous ornament across the body merely to create unity.

## Control Openings and Cutouts

Do not create an unusual garment opening or cutout merely to reproduce the element shape. A shaped hole in a single fabric layer usually lacks construction logic and should be replaced with an attached, layered, reinforced, or functional application.

Allow an element-shaped opening only when it has a clear purpose, such as:

- providing access to a pocket, closure, carried object, or underlayer
- allowing ventilation or movement where the garment genuinely needs it
- forming part of a secured multi-layer construction with a visible underlayer
- operating as a closable vent, flap, or adjustable opening

When an opening is justified, specify its edge finish, reinforcement, closure, underlayer, or operating method. Prefer a shaped pocket flap, reinforcement panel, fastening plate, adjustable tab, or layered overlay when it can provide the same visual effect without removing fabric.

## Plan Full-Body Deployment

Allocate a visual budget before designing garments:

- **Primary expression:** normally one prominent application that establishes the element.
- **Secondary expressions:** normally two or three applications that echo it through different methods.
- **Accent expressions:** a small number of restrained details such as embroidery, hardware, or hanging ornaments.
- **Implicit expressions:** optional structural or line-based references that do not show the complete motif.
- **Quiet areas:** deliberately undecorated areas that preserve contrast and prevent saturation.

Balance the upper and lower body as one composition:

- If the upper body carries the primary expression and several secondary expressions, keep the trousers restrained.
- If the upper body is simple, the trousers or lower-body accessories may carry more of the element system.
- Do not let the upper garment, trousers, footwear, and accessories all compete as primary expressions.
- Do not distribute details evenly across the body.
- Cluster related detail intentionally and preserve nearby quiet space.
- Prefer varied translations of the same element over repeated copies of the complete symbol.

Treat these counts as defaults, not arithmetic. Adjust them for restrained, balanced, or rich designs while preserving hierarchy.

## Element Quality Test

A useful core element should:

1. remain recognizable through more than one transformation
2. support at least two different application methods
3. produce at least one application that combines decoration with practical construction or use
4. improve decisions for multiple clothing or accessory locations
5. help reject unrelated details
6. remain present after small badges, tools, and minor ornaments are removed

If an element can only appear as one isolated print or decoration, treat it as a minor motif rather than the organizing core element.

## Failure Conditions

Revise the element system when:

- the selected element is still only a broad theme or abstract adjective
- the complete symbol is copied repeatedly across every garment
- element applications ignore the function or construction of their locations
- decorative pseudo-function adds unnecessary pockets, straps, clasps, mounts, or protective pieces
- a single-layer garment receives an arbitrary element-shaped hole or cutout
- an unusual opening lacks reinforcement, closure, underlayer, access purpose, ventilation purpose, or movement purpose
- the design contains no stable recognition anchors
- every body region has equal visual weight
- upper and lower garments compete without hierarchy
- accessories merely duplicate already sufficient garment decoration
- accessory applications default without reason to waist-mounted small objects
- a conspicuous chest application floats without visible support or lacks a strong character, structural, or compositional basis
- high-visual-weight ornaments become routine merely because the clothing is minimal or monochrome
- removing small decorations removes the entire element system
- the deployment plan leaves no quiet areas
